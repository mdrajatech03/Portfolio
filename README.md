# 📷 Web Camera 3D Viewer

A simple web project that uses the **device camera** to display a live camera feed and render a **3D object** on the screen.  
This project is built using **HTML, CSS, and JavaScript** and works directly in modern browsers.

---


https://mdrajatech03.github.io/4D/

## 🚀 Features

- Access device **camera using WebRTC**
- Render **3D object visualization**
- Works on **mobile and desktop browsers**
- Lightweight and easy to use
- No external installation required

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript
- WebRTC (getUserMedia API)
- WebGL / Three.js

---

## 📱 How to Run the Project

### Method 1: Open Locally
1. Download or clone the repository.
2. Open the project folder.
3. Run the `index.html` file in your browser.

⚠️ Note: Some browsers may block camera access when opened directly from local files.

---

### Method 2: Run Using GitHub Pages (Recommended)

1. Upload the project to GitHub.
2. Go to **Repository Settings → Pages**.
3. Enable **GitHub Pages**.
4. Open the generated link in your browser.

Camera permission will be requested automatically.

---

## 🔐 Permissions Required

This project requires the following permission:

- **Camera access**

When the page loads, the browser will ask for camera permission.  
Click **Allow** to enable the camera feed.
