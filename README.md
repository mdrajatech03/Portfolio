# 🚀 Professional Portfolio Website

Welcome to my personal portfolio repository! This project showcases my skills, projects, and experience as a developer in a sleek, responsive, and interactive way.

## ✨ Features

**Fully Responsive:** Optimized for all screen sizes (Mobile, Tablet, and Desktop).

**Interactive UI:** Smooth transitions and animations using JavaScript.

**Project Filtering:** Integrated with MixItUp.js for seamless category filtering in the portfolio section.

**Clean Design:** Modern aesthetic with a focus on readability and user experience.

**Contact Form:** A functional layout for potential clients or employers to reach out.

## 🛠️ Tech Stack

**HTML5:** Structure and Semantic markup.

**CSS3:** Custom styling and Flexbox/Grid layouts.

**JavaScript (ES6+):** Dynamic functionality and DOM manipulation.

**MixItUp.js:** For high-performance filtering of portfolio items.

## 📂 Project Structure

├── index.html        # Main landing page
├── style.css         # Custom styles
├── script.js        # Main logic and animations
├── mixitup.min.js    # Filtering library
└── assets/           # Images (1.jpg to 6.jpg, etc.)

## 🚀 How to Run Locally

**1. Clone the repository:**
   
   git clone https://github.com/mdrajatech03/Portfolio.git
   

**2. Navigate to the folder:**

   cd Portfolio

**3. Open index.html** in your favorite browser.
