Project Overview

This project is a Bluetooth-controlled robot car designed using Arduino. It allows users to control the car's movement (Forward, Backward, Left, Right) wirelessly through a smartphone application. It uses an HC-05 Bluetooth module for communication and an L298N motor driver to control the DC motors.

Key Features

  • Wireless range of up to 10 meters.
  • Real-time response with zero latency.
  • Dual speed control modes.
  • Obstacle detection capability (optional upgrade).
